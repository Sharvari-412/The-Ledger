# The Ledger

A ledger-styled expense tracker with a running balance and savings goal tracking, built with Next.js + Supabase, in INR.

**Start here → [MANUAL.md](./MANUAL.md)** — full step-by-step setup, local run, deploy to Vercel, and troubleshooting.

Quick start if you already have the Supabase project and `.env.local` set up:
```bash
npm install
npm run dev
```
